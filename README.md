# 🎮 Simon Says Game
A simple and interactive **Simon Says memory game** built using JavaScript.  
This project focuses on core frontend concepts like event handling, DOM manipulation,Keyboard Events, and game logic.

---
## 🚀 Features
- 🎯 Sequence-based memory game
- ⌨️ Game starts on **any key press** (Keyboard Event Listener)
- 📈 Increasing difficulty with each level
- 🏆 **High Score tracking**
- 💡 Simple and clean UI
- ⚡ Fully browser-based (no installation needed)
---
## 🛠️ Concepts Used
- JavaScript Event Listeners
- Keyboard Events
- DOM Manipulation
- Game Logic Implementation
- Random Sequence Generation
---
## 🎮 How to Play
1. Press **any key** on your keyboard to start the game  
2. Watch the pattern of colors carefully  
3. Repeat the same sequence by clicking the buttons  
4. With each level, a new step is added  
5. If you make a mistake, the game ends  
6. Game over and your **high score** is shown to restart game again press any key from keyboard.
